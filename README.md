# rustlike-dungeons
![Alt text](screenshots/title.png?raw=true "Title")  
roguelike game written in rust

## features
- Keyboard and mouse interface
- Level up system
- Different monsters
- Spells & Potions
- Equipment
- Save game
- Dynamic FOV
- Dungeon level progression
- And more!

## dependencies
- tcod
- rand
- serde
